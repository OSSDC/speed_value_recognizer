# Speed Value Recognizer

# Requirement:
* OpenCV
* Tesseract

## Usage:
* Clone repository   
* Create build directory:   
`$ mkdir _build`   
`$ cd _build`
* Configure project:   
`$ ccmake ..`   
* Press `c` two times and after generate with `g`   
* Build:   
`$ make`   
* Run example:   
`$ cd ..`  
`$ ./_build/read_text ./img/number.png`  
* Output:
`Result: 82685`
